ABOUT TECHNOCRAT

Technocrat is an ambient techno machine based upon the principle that everything should arpeggiate upwards. It moves gradually between crisp melodies, twinkling drones, and fuzzy noise.

TIPS
  • No sound? Use Firefox 4, or a Canary build of Chrome with the Web Audio API enabled.
  • Still no sound? Wait a while. Everything changes slowly.
  • Change one thing at a time. Listen for the results.
  • Can't work out what a control does? Wait a bit longer.
  • Inaction is normally better than action.
  • Some controls are more subtle than others. Listen closely for changes.
  • Make sure you know which channel you are modifying. Look at the indicator above the sliders.
  • Have fun!
